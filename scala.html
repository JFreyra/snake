Welcome to Scala

What is Scala?

Scala is short for "Scalable Language." You can play with it on a small level, testing one line of code at a time to observe the effects, or you can build massve projects and run large systems based on Scala like LinkedIn, Twitter, Foursquare, Sony, and Lichess.

Scala syntax is concise and intuitive; its types aren't a hassle because the compiler can assume what they are. Despite its apparent simplicity, Scala is the preferred language for many server systems. The generated code is on the same level as Java's and its precise typing means that many problems are caught at compile-time rather than after deployment.

At the root, the language's scalability is the result of a careful integration of object-oriented and functional language concepts.

Object Orientation and Functionality

Scala is a object-oriented language. Every value is an object and every operation is a method-call. The language supports traditional design patterns like visitors and singletons. Scala also allows for the addition of new operations on classes that have been predefined like data types since every value is essentially an object. This is done through Implicit Classes. Through Interopability, this practice works across any class in Scala and Java.

Scala is also a full-blown functional language. It has first-class functions, a library with efficient immutable data structures, and a general preference of immutability over mutation.

You can start to use it as a "Java without semicolons". Over time, you can progress to gradually an immutable state in your code, phasing in functional coding patterns. This progression is often a good idea. At the same time, Scala is flexible and easy to work with in any style.

Compatibiltiy

Scala allows for an easy transition from Java and other computing languages you may already know. Not only does the language allow you to gradually adjust, Scala is made so that it can be used freely along with Java clases. The use of the Java compiler within the Scala compiler allows for this.

It runs on most JVMs and can make use of the large pool of Java libraries, framworks, and tools that already exist. Scala frameworks, such as Akka, Finagle and the Play web, offer dual APIs for Java and Scala as well.

Language Constructs and Growth

What makes Scala unique is its language setup and organization. By developing objects in a flexible manner, they can be easily combined. Functions and mathematical data types are defined within classes. Scala also supports pattern matching for arbitrary classes.

A set of 20-50 basic methods allow the user to master most problems allowing them to rely less on mutable states. These methods also allow higher abstraction levels of organization with lightweight syntax.

Scala then utilizes these constructs and applies them with scalable server software. The language allows the user to write high-performing multi-threaded code. It uses concurrent and synchoronous processing and parallel use of several cores.
